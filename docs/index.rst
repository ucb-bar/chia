Welcome to CHIA's documentation!
================================

**CHIA**  is **an open-source framework for agile and principled 
hardware/software co-design research** 
— where humans, agents, and existing
hardware-design tools all work together.

New to CHIA? Start with the :doc:`/getting-started/chia-basics` page, or go
straight to :doc:`/getting-started/installation` to set things up.

The current release of CHIA is considered a beta.
If you find any issues, please reach out to us via the
`user forum <https://groups.google.com/g/chialoops>`_ or
`contact the lead developers / PI <mailto:angelacui@berkeley.edu,ferran@berkeley.edu,jtoubes@berkeley.edu,sagark@eecs.berkeley.edu>`_
(Angela Cui, Ferran Hermida-Rivera, Jack Toubes, and Sagar Karandikar).

CHIA stands for:

(**C**\ o-designing **H**\ ardware and software with **I**\ ntelligent **A**\ gents)

Originally CHIA stood for:

(**CH**\ ipyard's **I**\ ntelligent **A**\ gents)

(**CH**\ ipyard's **I**\ ntelligence **A**\ gency)
